substr(), substring(), slice() 비교

substring() VS slice()

  • start 인덱스가 end 인덱스보다 큰 경우
    • substring() : start 값과 end 값을 바꾸어 처리.
    • slice() : 비어있는 string("") 출력

  • 인덱스가 음수인 경우
    • substring() : 0으로 취급
    • slice() : 뒤에서 (음수 인덱스) 번째의 인덱스로 처리. 5자인 문자열에 slice(-2,5) 를 적용하면 slice(3,5) 로 처리된다.
      단, 음수 인덱스값이 문자열의 길이보다 클 시 0으로 처리된다.
