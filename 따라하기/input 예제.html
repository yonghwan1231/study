사용자로부터 색깔 입력받기

가장 좋아하는 색깔 :

input type='file' 속성 알아보기 ( 파일 입력 )

accept 속성 : 첨부파일의 유형 지정
ex) image/* 는 모든 유형의 이미지 파일을 뜻함
ex) image/jpg 는 jpg유형의 이미지 파일을 뜻함 ex) image/*,.txt 는 모든 유형의 이미지 파일과 txt 파일을 뜻함(.txt처럼 마치표로 시작하는 확장자명을 입력)

capture 속성 : accept 속성에 이미지 또는 비디오를 입력받는 경우, 기기의 어떤 카메라를 이용할지 지정한다. user는 전면카메라, environment는 후면카메라.

files 속성 : 사용자가 파일을 선택하면, 선택된 파일의 목록이 FileList 객체 형태로 flies 속성에 저장된다.

multiple 속성 : 사용자가 ctrl / shift 키를 사용하여 한번에 여러개의 파일을 선택하게 할지, 한번에 하나의 파일만 선택하게 할지를 결정하는 속성. boolean 값을 가진다.

required 속성 : submit 버튼으로 입력값을 전송할때, 반드시 파일이 선택 되어야 한느지 여부를 지정하는 속성이다. boolean 값을 가진다.

value 속성 : 사용자가 선택한 파일의 경로값을 가진다. 만약, 여러 파일을 선택한 경우, 목록의 첫번째 파일 경로값만을 가진다. 단, 악의적인 사용 방지를 위해 모든 파일의 겨올 앞에 'C:'fakepath\'가 포함된다.
